# **App Name**: SS Smart Haat

## Core Features:

- Homepage Hero Section: A modern hero section to showcase featured products and promotions.
- Product Listing: Display products in a visually appealing card format with image, name, and price.
- Product Details Page: Dedicated page for each product with detailed information and larger images.
- Responsive Design: Fully responsive layout that adapts to different screen sizes (mobile and desktop).

## Style Guidelines:

- Primary color: Soft beige (#F5F5DC) for a neutral and sophisticated base.
- Background color: Very light gray (#F0F0F0), almost white, for a clean backdrop.
- Accent color: Muted gold (#C0B283) for highlights and calls to action.
- Body and headline font: 'Inter' sans-serif for a modern, neutral, and readable feel.
- Clean and minimalist layout to emphasize products.
- Subtle transitions and hover effects to enhance user experience.
